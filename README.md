# Permuted MNIST — Softmax Linear Agent

Linear softmax classifier (logistic regression) using PyTorch, evaluated on the [permuted MNIST environment](https://github.com/ml-arena/permuted_mnist).

## Install

```bash
uv pip install "git+https://github.com/ml-arena/permuted_mnist.git" torch
```

## Run

```bash
uv run python eval.py
```
